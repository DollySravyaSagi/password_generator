# Password Generator

A simple and interactive password generator built using HTML, CSS, and JavaScript. This project allows users to generate strong and secure passwords based on selected criteria.

## Features

* Generate random secure passwords
* Customize password length
* Include:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* Copy password to clipboard
* Clean and user-friendly UI

## Tech Stack

* HTML
* CSS
* JavaScript

## Project Structure

```
project-folder/
│── index.html
│── style.css
│── script.js
```

## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Generate your secure password 🚀
